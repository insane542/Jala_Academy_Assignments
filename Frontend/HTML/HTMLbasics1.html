The BMW M3 GTR is a high-performance version of the BMW 3 Series, developed by BMW's in-house motorsport division, BMW M GmbH. M3 models have been produced for every generation of 3 Series since the E30 M3 was introduced in 1986.

Key Features

  1. 0-60 mph in 3.4 seconds
  2. 493 horsepower
  3. 4.0-liter V8 engine
  4. Limited production run
